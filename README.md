# Base64-Converter
A JavaScript converter for base64

Just a small side project for me to learn how base64 encoding and decoding works

## Encoding

### Input:

```bash
node ./base64-converter.js encode Hello, my name is Quin
```

### Output:

```
SGVsbG8sIG15IG5hbWUgaXMgUXVpbg==
```

## Decoding

### Input:

```bash
node ./base64-converter.js decode SGVsbG8sIG15IG5hbWUgaXMgUXVpbg==
```

### Output:

```
Hello, my name is Quin
```
